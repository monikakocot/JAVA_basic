# JAVA_basic
Excercises for learning basic JAVA

In this repository I put my basic "applications" that I did to learn the basics of Java.
Mainly the exercises concern the basic issues:

- Loops,
- Methods,
- Class scanner,
- Tables,
- Collections,
- Associations,
- Inheritance,
- Polymorphism,
- Interface,
- Abstract classes,
- Object oriented design.
